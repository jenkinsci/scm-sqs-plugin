Name oder URL einer Nachrichtenwarteschlange die im Amazon Simple Queue Service (SQS) angelegt wurde. Das Plugin startet Builds wenn Nachrichten in dieser Queue abgelegt werden.

Die Queue wird nicht aktiv überwacht, bis mindestens ein Job so konfiguriert wurde, dass er auf Nachrichten aus dieser Queue horcht. Siehe Build Trigger Konfiguration von Jobs.

Weiterführende Dokumentation ist im SCM SQS Plugin Repository auf GitHub verfügbar.
